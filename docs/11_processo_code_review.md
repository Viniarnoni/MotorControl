# 11 - Processo de Code Review

**Projeto:** MotorControl
**Versão:** 1.0

## 1. Objetivo
Definir as regras de revisão de código por pares antes de integrar as alterações no branch principal.

## 2. Conteúdo Principal
* **Critérios de Revisão:** Todo código deve ser verificado em relação às travas da RN03 (limite de desconto de 15%) e uso correto do SQLModel.
* **Aprovação:** É necessário o aceite de ao menos 1 engenheiro revisor para validar o Pull Request.
