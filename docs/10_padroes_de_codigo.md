# 10 - Padrões de Código (Style Guide)

**Projeto:** MotorControl
**Versão:** 1.0

## 1. Objetivo
Estabelecer os padrões de formatação PEP 8 para manter o código Python limpo e uniforme.

## 2. Conteúdo Principal
* **Indentação:** Obrigatório o uso de 4 espaços (nunca Tabulações).
* **Nomenclatura:** Classes usam PascalCase, funções e variáveis usam snake_case.
* **Componentes Flet:** Devem explicitar o tipo no sufixo, ex: btn_salvar, txt_nome.
